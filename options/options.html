Subtext Settings

Manage your Subtext data.

Subtext sends article text to its analysis server when you request an analysis. Results are cached in session storage so the current browsing session can restore previous results.

Data handling

  • Article title, URL, source, excerpt, and article text are sent to the Subtext analysis server to generate a summary and language signal indicators.
  • Cached analysis results are stored in session storage so the current browsing session can restore previous results.
  • You can clear cached analysis data from this page.
Privacy policy

Free tier

  • 15 analyses per day, per device.
  • Your daily count resets at midnight UTC.
  • Previously analyzed articles load from cache and count toward your daily limit.
