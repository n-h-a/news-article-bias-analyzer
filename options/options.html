Subtext Settings

Control how analysis data is handled.

Subtext validates your saved API configuration with OpenAI in Settings and sends article text to OpenAI when you request an analysis. Your API key is stored in extension storage on this browser until you remove it.

API configuration

Subtext uses your API key directly from this extension. Your OpenAI account is billed for any analysis requests made with that key.

Data handling

  • Article title, URL, source, excerpt, and article text are processed to generate a summary and bias indicators.
  • Cached analysis results are stored in session storage so the current browsing session can restore previous results.
  • You can remove the stored API key and clear cached analysis data from this page.
